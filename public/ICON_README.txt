# Icon Placeholder

Für das finale Build sollte hier ein echtes Icon platziert werden.

## Benötigte Formate:
- icon.png (512x512 PNG)
- icon.ico (Windows Icon mit mehreren Größen: 16x16, 32x32, 48x48, 256x256)

## Design-Vorschlag:
- Modern, minimalistisch
- Farben: Accent-Green (#00ff88) auf dunklem Hintergrund
- Symbol: Stilisiertes "J" oder Roboter-Kopf
- Cyber/Tech-Ästhetik passend zur UI

## Tools zum Erstellen:
- PNG to ICO: https://convertio.co/png-ico/
- Icon Generator: https://www.favicon-generator.org/

## Platzierung:
- public/icon.png
- public/icon.ico

Für den Entwicklungsmodus wird ein Standard-Electron-Icon verwendet.
